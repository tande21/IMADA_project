Allgemeine Hinweise:

Wird im Browser ein Zoom-Level ungleich 100% oder in den Betriebsystemeinstellungen eine Bildschirm-Skalierung verwendet kann es zu Problemen mit der Mausbedienung kommen.
Es wird empfohlen, das Spiel im Vollbild-Modus zu spielen. Dazu auf den Vollbild-Button unterhalb des Spiele Fensters drücken, nachdem das Spiel geladen hat.
Teilweise verliert das Spiel-Fenster den Fokus, wodurch man sich nicht mehr frei umschauen kann. Um das Problem zu beheben, einmal in das Spiel-Fenster klicken.
